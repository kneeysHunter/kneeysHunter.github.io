## 题目描述

给你一根长度为n的绳子，请把绳子剪成整数长的m段（m、n都是整数，n>1并且m>1，m<=n），每段绳子的长度记为k[1],...,k[m]。请问k[1]x...xk[m]可能的最大乘积是多少？例如，当绳子的长度是8时，我们把它剪成长度分别为2、3、3的三段，此时得到的最大乘积是18。

## 输入描述:

```
输入一个数n，意义见题面。（2 <= n <= 60）
```

## 返回值描述:

```
输出答案。
```

示例1

## 输入

```
8
```

## 返回值

```
18
```

- 思路一 动态规划 不建议用
- 思路二，3越多 乘和的值越大。（只能说牛逼）

```python
# -*- coding:utf-8 -*-
import math
class Solution:
    def cutRope(self, number):
        sum = 1
        if number<4:
            return number
        if number==4:
            return 4
        while number>4:
            sum *= 3
            number-=3
        return sum*number
            
        
```
