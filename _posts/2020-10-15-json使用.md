#  JSON 一种轻量级的数据交换格式

##  JSON文档结构中 有两种基本的结构，对象和数组

规范格式，名称和数据都必须双引号

- 对象采用      <em>‘名称：值’<em>   

  - ```json
    {
        "name" : "ahdsoih"
        "size" : 345
        "saved”：true
    }
    ```

- 数组是	<em>一连串元素的集合<em>

  - ```json
    ["text","html","css"]
    ```
